Cos'è bootstrap

Bootstrap è un framework front-end gratuito per uno sviluppo Web più rapido e semplice, include modelli di progettazione basati su HTML e CSS e ti dà anche la possibilità di creare facilmente progetti reattivi

Cosa devi importare per creare una pagina web con bootstrap

Per creare una pagina web con bootstrap bisogna importare la riga di codice <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css">

Cos’è un CDN? Che vantaggi comporta rispetto a scaricare bootstrap sul tuo sito?

A cosa serve questo attributo del tag meta: content= width=device-width, initial-scale=1

width=device-width serve a impostare la larghezza della pagina in base al dispositivo che si sta utilizando, mentre initial-scale = 1 imposta il livello di zoom iniziale

Che differenza c’è tra Container e Container Fluid?

il Container è a larghezza fissa, mentre il Container Fluid adatta la propria larghezza alla pagina
